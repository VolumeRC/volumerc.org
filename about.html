VolumeRC

VolumeRC is an experimental open source project for the development of Web visualization technologies for volumetric objects, without the use of external plugins. This kind of objects are widely used in several science areas, such as medicine, geology, meteorology, etc., but the inherent visualization difficulties and the huge amount of computing resources required made not possible for this methodology to spread.

Nowadays, the implantation of new standards, such as HTML5 and WebGL, allows harnessing the capacity of the graphics cards; thus, amazing views can be generated using web technology on different frames.

Hereby, this website intends to centralize efforts on the development of web volumetric images, focusing on several projects, like an example of a library for basic volume visualization in javascript, aside from diverse wrappers in other architectures or graphic user interface web systems, such as jquery.

Finally, Vicomtech-IK4 participates together with different standardization groups presenting this technology in the widest possible way. For this reason, there is a collaboration established with the Web3D consortium, particularly with the medical visualization group. VolumeRC technology will be used as baseline for the development of the future MEDX3DOM standard. Besides, Vicomtech-IK4 collaborates with the German Fraunhofer IGD institute.
